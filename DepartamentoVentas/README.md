# Departamento de Ventas

Entorno virtual para análisis de datos del departamento de ventas.

## Requisitos
- Python 
- pip

## Instalación
```bash
python -m venv DepVentas
DepVentas\Scripts\activate
pip install -r requirements.txt
```

## Estructura
```
Departamento_Ventas/
├── DepVentas/
├── src/
├── notebooks/
├── data/
├── requirements.txt
└── README.md
```
